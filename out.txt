### README.md

```markdown
# repo-to-llm

CLI tool to convert a code repository into LLM-friendly markdown.

## Installation

```bash
pip install repo-to-llm
````

## Usage

```bash
repo-to-llm /path/to/your/repo > output.md
```

## Contributing

Contributions are welcome. Please open issues or pull requests on GitHub.
